# Getting Started

# Reservation System
## Tech stacks
 * Gradle 8.10.2
 * Spring boot 3.3.4
 * Java 17
 * H2 Database
 * Openapi

# IDE
 preferred IDE is intellij community edition.

## Run the app
./gradlew clean build 

## H2 database console 
>http://localhost:8080/h2-console

## Open Api Docs
> http://localhost:8080/swagger-ui/index.html
